# AI Disclosure Form

## Tool: ChatGPT

### What did you use it for?
ChatGPT was used throughout the project as a support tool rather than something that built the entire app for me. I mainly used it for debugging errors, understanding concepts, improving code structure, and writing documentation. It also helped with architectural decisions, such as how to separate frontend and backend logic, how to organize folders, and how to structure API routes more cleanly.

In some cases, I used it to refactor parts of my code when things became messy or repetitive. For example, I asked for suggestions on how to simplify API routes, improve readability, and make the code more modular instead of having everything in one place.

### Example prompt:
"How can I refactor my Express API routes to make them cleaner and more modular?"

## Tool: GitHub Copilot

### What did you use it for?
GitHub Copilot was mainly used during development for speeding up coding. It helped by suggesting code as I typed, especially for repetitive parts like API routes, function structures, and basic frontend logic.

It was useful for saving time, but I did not rely on it blindly. I always reviewed the suggestions and adjusted them to match my project requirements.

While writing backend routes, Copilot suggested function structures and request handling logic that I then modified and cleaned up.

## Overall Reflection

Using AI significantly improved my workflow, especially when debugging and restructuring code. It helped me move faster when I was stuck and gave me ideas on how to organize my project more effectively. For example, I used AI to improve the architecture of my application by separating concerns between frontend, backend, and database logic instead of keeping everything tightly coupled.

However, I still had to understand and modify everything that was generated. AI did not replace my thinking — it supported it. In many cases, I had to refine prompts multiple times to get useful outputs, and I often rewrote parts of the suggestions to better fit my project.

Without AI, the project would have taken longer, especially when debugging errors or writing documentation, but I believe I would still be able to build the core functionality. AI mainly helped improve speed, structure, and clarity.
